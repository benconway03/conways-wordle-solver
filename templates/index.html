How to Use

• Start by pressing the Submit button with both boxes left blank

• Type the outputted 5-letter word into Wordle and note the colour pattern you got back

• Type the word you tried in Wordle into the top box of the solver and in the bottom box, type the colour pattern (Grey = X, Yellow = Y, Green = G)

• Then press Submit, type the outputted word into Wordle and repeat

• To reset the memory, so that the website forgets previous words you've tried and their results, press Submit with two blank boxes
